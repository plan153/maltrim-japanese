📹 촬영 · 편집 기술
화면 녹화 해상도
1080×1920 이상 (9:16). 60fps 권장. 시스템 UI (배터리, 알림) 숨김 처리.
실사 촬영 기기
iPhone 또는 동급 스마트폰 후면 카메라. 핸드헬드, 안정적으로. 짐벌 권장.
편집 소프트웨어
Premiere Pro / DaVinci Resolve / CapCut (간편). 자막은 별도 레이어로.
전환 규칙
씬 02~05 사이는 Hard Cut 원칙. 씬 06·07 진입 시만 슬라이드/페이드 허용.
색 보정
전체 채도 약간 낮춤. 파란 계열 강조. 화면 화이트발란스는 자연스럽게.
🎙 내레이션 · 음성
성별 / 연령대
30~40대 여성 또는 남성. 선생님 같은 차분하고 친근한 톤. 과하게 밝지 않게.
속도
일반 대화보다 약 10% 느리게. 강조 단어에서 한 박자 쉬기.
강조 단어
「당황하지 않아도」 「보기 버튼」 「허용」 「꾹 누르고」 「85점 이상」 「어미만 바꾸면」 — 이 6개는 볼드 발음.
녹음 환경
방음 처리된 공간. 에코 제거 플러그인 사용. 마이크는 콘덴서 마이크 권장.
🎵 배경음악
장르 / 분위기
피아노 + 경전타악 (로파이 분위기). 안심감. 너무 감성적이지 않게.
템포
80~90 BPM. 씬 06부터 살짝 업템포 전환 권장.
볼륨 레벨
씬 01~05: VO 대비 40% / 씬 06~07: 50~60% / 마지막 3초: 페이드아웃
추천 소스
Epidemic Sound「Warm Piano」 계열, Artlist 로파이 피아노 카테고리. 상업 라이선스 확인 필수.
