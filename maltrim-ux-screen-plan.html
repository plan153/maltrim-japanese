목표 선택
목표 (Goal)

학습 동기를 설정하고, 마이크 권한을 자연스럽게 허가받아 첫 세션까지 진입 장벽을 제거한다.

주요 동작

  • 스플래시 → 슬라이드 3장 (Why · How · What)
  • 학습 목표 선택 (여행·업무·문화·시험)
  • 하루 목표 시간 선택 (5·10·15분)
  • 마이크 권한 허가 (이유 설명 선행)
  • 닉네임 입력 (선택, 건너뛰기 가능)

UI 요소

  • 풀스크린 슬라이드 (Lottie 애니메이션)
  • 선택 카드 그리드 (활성화 테두리)
  • 하단 고정 CTA 버튼
  • 진행 도트 인디케이터
  • 권한 설명 바텀시트
상태 처리
빈 화면: 선택 전 → CTA 비활성(회색) 로딩: 선택 저장 중 → 스피너 오류: 마이크 거부 → 인앱 안내 배너
CTA
「시작하기」→ 슬라이드 1 「다음」→ 단계 진행 「건너뛰기」→ 닉네임 스킵 「지금 바로 연습하기」→ 홈
마이크로카피
🎙 "발음을 들려줘야 정확히 교정해 드릴 수 있어요. 마이크는 연습할 때만 켜져요."
✈️ "3분이면 오늘 배울 문장이 준비돼요!"
건너뛰기 링크 텍스트: "나중에 설정할게요" (버튼 아님 → 탭 영역 명확히)
사용자 유지 기회
🔒 권한 허가 직후 "마이크 테스트 해볼까요?" 인라인 미니 녹음 → 즉각 성공 경험 제공으로 이탈 방지

목표 (Goal)

매일 앱을 열었을 때 "오늘 뭘 해야 하는지" 즉시 파악하고, 1탭으로 학습 시작.

주요 동작

  • 오늘 복습 + 신규 문장 카운트 확인
  • 연속 학습 스트릭 뱃지 확인
  • 30일 달력에서 완료일 시각화
  • 「오늘 학습 시작」→ 발음 연습 화면
  • 탭바: 홈·단어장·통계·설정

UI 요소

  • 상단 인사 헤더 (시간대별 문구)
  • 스트릭 뱃지 + 화염 애니메이션
  • 오늘 세션 카드 (복습 N개 + 신규 N개)
  • 30일 진도 미니 달력
  • 하단 탭바 (4탭)
상태 처리
빈 화면: 첫날 → "오늘의 첫 문장을 만나볼까요?" 대형 CTA 로딩: 스케줄 API → 스켈레톤 카드 오류: 오프라인 → "오프라인 모드: 저장된 문장 연습"
완료: 오늘 학습 끝 → 완료 애니메이션 + 내일 예고
CTA
「오늘 학습 시작 →」(대형 원색 버튼) 「복습 먼저」/ 「신규 먼저」스위치 「통계 보기」→ 통계 화면
마이크로카피
오전: "おはようございます! 오늘도 화이팅이에요 ☀️"
완료 후: "오늘 할 건 다 했어요! 내일 또 만나요 🎌"
스트릭 위험: "내일 놓치면 🔥가 꺼져요. 딱 2분만요!"
사용자 유지 기회
📬 스트릭 위험 시 (18시 이후 미학습) 푸시 알림 + 앱 배지. 완료 시 "내일 미리보기" 카드로 내일 재방문 앵커 제공.

목표 (Goal)

원어민 TTS를 듣고 즉시 따라 말하는 반복 루프로 조건반사적 발화 습관을 형성한다.

주요 동작

  • 일본어 문장 카드 표시 (독음·의미 blur)
  • 🔊 → Neural TTS 재생 (0.85배속)
  • 🎤 홀드 또는 자동 녹음 (3초)
  • 실시간 STT 텍스트 스트림
  • LCS diff 채색 피드백 표시
  • 85점↑ → 자동 다음 / 미만 → 락 루프
  • L&R 모드 (3회 연속 평균 모달)

UI 요소

  • 대형 일본어 텍스트 (28pt+)
  • 독음/의미 블러 토글 버튼
  • TTS 버튼 · 마이크 버튼 · 힌트 버튼
  • 진행 바 (오늘 N/M)
  • 파형 애니메이션 (녹음 중)
  • FFD 반응속도 칩 (⚡/🐢)
  • VAD 머뭇거림 칩
  • 오답 교정 말풍선 레이어
상태 처리
대기: 문장 카드만 표시, 마이크 비활성 TTS 재생 중: 마이크 버튼 회색 녹음 중: 파형 + 카운트다운 링 STT 오류: 토스트 + 재시도 버튼
85점↑ 통과: 초록 링 + 진동 햅틱 + 자동 전환 락 루프: 빨간 오버레이 + 1.5초 재시도
CTA
「🔊」→ TTS 재생 「🎤」홀드/탭 → 녹음 「💡」→ 힌트 말풍선 「→」→ 다음 문장 (통과 후) 「✕」→ 세션 종료 확인
마이크로카피
녹음 전: "TTS를 듣고 따라 말해보세요"
통과: "⚡ 조건반사 마스터! 반응속도 0.7초"
락 루프: "조금만 더! 85점이면 통과예요 💪"
STT 거부: "🔒 마이크 권한을 허용해 주세요" → 설정 이동 버튼
사용자 유지 기회
🎮 SVG 링 게이지 + 제목 피드백(조건반사 마스터/유창성 우수)으로 세션 중 지속 몰입. 3회 연속 95점↑ 시 "완벽 마스터!" 특별 애니메이션 → SNS 공유 버튼.

세션 완료 결과
목표 (Goal)

오늘 학습 성취를 시각화해 보상감을 극대화하고, 명확한 다음 행동(홈/공유/이어서)을 제시한다.

주요 동작

  • 세션 평균 점수 SVG 게이지 애니메이션
  • 반응속도·연습 수·스트릭·신기록 4격자
  • 오늘 틀린 문장 diff 미리보기
  • 에빙하우스 다음 복습일 예고
  • SNS/카카오 공유 버튼
  • 「이어서 자유 연습」 / 「홈으로」

UI 요소

  • 대형 SVG 링 게이지 (점수 애니메이션)
  • 2×2 성취 격자 카드
  • 오류 diff 요약 (최대 3개)
  • 다음 복습 예정 배지
  • 공유 카드 미리보기 (이미지 생성)
  • Confetti 애니메이션 (90점↑)
상태 처리
무결점 세션 (오류 0): "완벽한 발음이에요! 🏆" 특별 화면 결과 집계 중: 게이지 0→N 카운트업 DB 저장 실패: 로컬 저장 + "다음에 동기화" 안내
CTA
「홈으로 →」(기본 버튼) 「이어서 연습」(보조 버튼) 「공유하기」(카카오/인스타) 「틀린 문장 다시 보기」(링크)
마이크로카피
90점↑: "🎉 오늘 최고 기록이에요! 내일도 기대돼요"
60~89점: "💪 잘 하고 있어요! 내일 이 문장 다시 만나요"
공유 카드: "[이름]님이 오늘 일본어 87점 달성! 🎌 #말트임"
사용자 유지 기회
📅 "다음 복습일: 내일 3개, 3일 후 5개 예정" → 미래 방문 앵커 설정. 스트릭 N일 달성 시 특별 뱃지 잠금 해제 → 프로필 화면 연결.

4지선다 퀴즈
목표 (Goal)

발음 연습 외 시간(통근·휴식)에 가볍게 어휘·문법 의미를 복습해 장기 기억을 강화한다.

주요 동작

  • 플래시카드: 앞(JP) → 탭 → 뒤(의미+독음)
  • 4지선다 퀴즈 (의미·독음·용례 교차)
  • 학습한 표현 전체 목록 필터(카테고리·난이도)
  • 북마크 저장 (오답 즐겨찾기)
  • TTS 버튼으로 발음 확인

UI 요소

  • 뒤집기 애니메이션 플래시카드
  • 4지선다 버튼 (정답 초록 / 오답 빨강)
  • 진행 바 + 정답률 칩
  • 카테고리 필터 탭 (여행·일상·음식…)
  • 검색 바 (JP/KO 텍스트)
상태 처리
미학습 상태: "아직 학습한 표현이 없어요 → 발음 연습 시작하기" 퀴즈 생성 중: 스켈레톤 카드 오프라인: 캐시된 표현으로 플래시카드 동작
CTA
「퀴즈 시작」(N개 준비) 「플래시카드」토글 「🔊」→ TTS 재생 「🔖」→ 북마크 「오답 다시 풀기」
마이크로카피
퀴즈 완료: "🎯 정답률 80%! 오답 5개는 내일 발음 연습에서 만나요"
오답 직후: "정답은 '~해 주세요'예요. 한 번 더 들어볼까요? 🔊"
사용자 유지 기회
🎰 퀴즈 완료 후 "오늘의 랜덤 표현" 카드 1장 제공 → 앱 체류 시간 연장. 오답 표현은 다음 발음 연습 세션에 우선 포함.

목표 (Goal)

초보자가 히라가나/가타카나 46자를 부담 없이 완주하도록 행(あいうえお)별 잠금 해제 방식으로 단계적으로 제공한다.

주요 동작

  • 행별 가나 그리드 (잠금 → 해제 진행)
  • 각 가나 탭 → 카드 뒤집기 (발음·예시어)
  • 행 완료 시 퀴즈 미니게임 (5문제)
  • 히라가나 ↔ 가타카나 탭 전환
  • TTS로 원어민 발음 청취

UI 요소

  • 5×10 가나 그리드 (잠금/완료/진행 색상)
  • 탭 뒤집기 카드 (가나·로마자·예시어)
  • 행별 진행 바
  • 히라가나/가타카나 세그먼트 컨트롤
  • 완료 뱃지 (행별 수집)
상태 처리
미시작: あ행만 열림, 나머지 잠금 처리 퀴즈 로딩: 스켈레톤 행 완료: 폭죽 + 다음 행 잠금 해제 애니메이션
CTA
「행 퀴즈 시작」→ 5문제 「🔊 발음 듣기」 「다음 행 열기」(완료 후) 「전체 복습」(46자)
마이크로카피
첫 행 완료: "あ행 마스터! 이제 か행이 열렸어요 🔓"
전체 완료: "히라가나 46자 완주! 이제 어떤 일본어든 읽을 수 있어요 🎌"
사용자 유지 기회
🔓 행 잠금 해제 진행 구조 → 완료 욕구 자극. 가나 완주 후 "이 문장도 이제 읽을 수 있어요!" → 발음 연습 화면 샘플 미리보기로 연결.

  • GitHub-style 학습 히트맵 (7×52)
상태 처리

목표 (Goal)

학습 경험을 개인화하고, 알림 설정으로 재방문율을 높이며, 프리미엄 업그레이드로 자연스럽게 유도한다.

학습 설정

  • 하루 목표 (5·10·15·30분)
  • TTS 속도 (0.7×~1.0×)
  • 학습 시작 화면 선택
  • 독음 표시 기본값

알림 설정

  • 매일 알림 ON/OFF
  • 알림 시간 (시간 피커)
  • 스트릭 위험 알림
  • 복습 예정 알림

계정·기타

  • 프리미엄 업그레이드
  • 데이터 내보내기
  • 진도 초기화
  • 앱 버전 · 문의
