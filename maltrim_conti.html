Scene 06 · CTA
"지금 도전하기" — 게임 시작 버튼처럼
📽 게임의 "START" 버튼처럼 연출. 로고 + "지금 도전하기" 버튼이 빛나는 네온 이펙트로 표시. 마지막 0.5초: 앱 첫 화면 플래시 컷 → 블랙아웃.

공통 — UX 배리어 해소 체크리스트
🌫️
흐릿한 텍스트 (blur)
첫 화면의 일본어 텍스트가 흐릿하게 표시됨. 화면 오류로 오해할 수 있음.
→ "의도된 설계, '보기' 버튼으로 확인" 명시
🎤
마이크 권한 팝업
브라우저 마이크 권한 요청이 갑자기 뜸. 거부하면 핵심 기능 비활성화.
→ "허용을 눌러야 AI 채점 가능" 선행 안내
👇
꾹 누르기 방식
마이크 버튼은 짧게 탭이 아닌 "누르고 있는 동안" 녹음. 짧게 탭하면 작동 안 함.
→ "꾹 누르기" 손가락 동작 직접 시연 필수
🔒
85점 락 화면
84점 이하 시 "🔒 다시 도전!" 화면이 뜸. 앱이 멈춘 줄 알고 종료할 수 있음.
→ "정상 작동, 85점 이상이면 자동 통과" 안내
🔁
3회 자동 녹음 모드
'듣고 따라하기' 버튼 누르면 TTS → 자동 녹음이 3회 반복. 예고 없이 시작돼 당황.
→ "총 3회 자동 반복, 준비 후 시작" 사전 설명
📅
Day 뱃지 탭 기능
상단 "Day 1/30" 뱃지가 탭 가능한 버튼임을 모름. 진도 이동 방법 불명확.
→ "위 뱃지 탭 → 30일 그리드 이동 가능" 안내
